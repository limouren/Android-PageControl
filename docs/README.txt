Android PageControl Library

A circular page indicator library for Android applications.

This library provides a customizable page control widget that can be used with ViewPager to show page indicators.

Features:
- Customizable indicator size and spacing
- Support for different indicator shapes (circle, rectangle)
- Configurable colors for current and normal states
- Clickable indicators for navigation

For full documentation and usage examples, please visit:
https://github.com/ksoichiro/Android-PageControl
